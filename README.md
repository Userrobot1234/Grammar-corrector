# Grammar-corrector
A lightweight AI-powered grammar checker built with Python and Transformers that automatically detects and corrects grammatical errors in text.
AI Grammar Checker

An AI-powered grammar correction tool built with Python and Hugging Face Transformers. The application analyzes user-entered text, detects grammatical errors, and generates corrected sentences in real time. It provides a simple and efficient way to improve writing quality without requiring Java or external grammar-checking software.

Features
Grammar and spelling correction
Real-time text analysis
AI-powered correction using Transformer models
Simple command-line interface
Java-free implementation
Built with Python and Hugging Face Transformers

Technologies Used

Python 3.10+
Transformers
PyTorch
SentencePiece
